# Images

Use this folder for portfolio images.

## Suggested structure

- `profile/` for your portrait or headshot
- `awards/` for award photos or certificates
- `projects/` for lab, hardware, or project visuals

## Current placeholder

The site currently uses:

`assets/images/profile/thevindu-portrait-placeholder.svg`

If you want to use a real photo instead, place it in `assets/images/profile/` and update the
`src` of the portrait image in `index.html`.
